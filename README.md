## Setup

### Dependencies

Install

```shell
pip install -r requirements.txt
```

Update

```shell
pip freeze > requirements.txt
```

### Virtual Env

Setup

```shell
python3 -m venv venv
```

Activate

```shell
source venv/bin/activate
```

Deactivate

```shell
deactivate
```

## Test

```shell
pytest
```
